# E-commerce Application  

This is a full-stack E-commerce application built with Node.js, Express, MongoDB, and React. The application allows users to register, log in, and provides a basic structure for managing e-commerce functionalities.

## Features  

- User registration and authentication  
- User login/logout  
- JWT-based authentication  
- Basic user interface with React  
- MongoDB for data storage  

## Tech Stack  

- **Frontend:** React.js  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** JWT, bcryptjs  
- **Environment Variables:** dotenv  

## Getting Started  

### Prerequisites  

To get started, ensure you have the following installed:  

- [Node.js](https://nodejs.org/) (v14 or later)  
- [npm](https://www.npmjs.com/get-npm) (comes with Node.js)  
- [MongoDB](https://www.mongodb.com/try/download/community) (you can use MongoDB Atlas for a cloud-hosted solution)  
- Git (optional, for cloning the repository)
